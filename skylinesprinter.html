Skyline Sprinter is a game inspired by games like Temple Run and Subway Surfer. The goal is to survive as long as possible by avoiding obstacles. There are also coins to collect for points.

This game features random generation of the world. The way this is implemented is by spawning premade city blocks in the player's path. As the player progresses the blocks are destroyed behind them. The blocks are organized as nodes with children.

If I were to continue work on this game one of the main things I would add is more city blocks with more variation. I would also want to exchange the player model. I got this one for free but I don't think the running animation looks very good. I also think the menus could use some work so they aren't so plain.
