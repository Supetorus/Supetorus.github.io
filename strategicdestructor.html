Strategic Destructor

GitHub

Itch.io

Strategic Destructor is a game built with Unity Engine inspired by Crush the Castle. The idea behind the game is that you have to knock down structures, but you have to do so strategically because some parts must remain standing. This game is a prototype and has a lot of room to grow.

The player can move their camera around the structure 360 degrees, as well as adjusting the up-down / left-right aiming of the canon. There are several camera modes available by pressing tab. The first camera mode is a view from directly behind the canon. The second lets the camera orbit the structure. The last view is an orthographic view from the side which makes it feel like you are playing a 2D game like crush the castle (although functionally it is kind of useless as it is impossible to tell what your shots will hit).
