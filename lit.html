LIT Engine is my capstone project at Neumont College of Computer Science. It started out as a rendering engine to learn about how 3D models are drawn to the screen. This project will continue to grow as I use it to learn about other game engine technologies. â€‹

I chose to use Direct3D11 as my rendering API because I heard that OpenGL is outdated, but looking back I would have been able to accomplish more if I had used OpenGL because the learning resources seem to be more plentiful and beginner friendly.

The editor functionality of this project may not be very impressive. The aim of this project was primarily to learn the technologies. Having something impressive to an end user or casual observer was secondary. I put a large emphasis in my own work on correctness of code. I didn't want to hack something together to look good, but rather to have high quality code that could be easily modified and built upon.

The project is written in C++, with HLSL shaders.

Features

  • Loads most standard model file types using assimp.
  • Live scene editing (translating, rotating, and scaling models as well as moving and rotating the camera), saving, and reloading.
  • Realtime lighting.
  • Loads and saves scenes in yaml using yaml-cpp.
  • Loads most standard image formats for texturing using stb_image.
  • Skyboxes
